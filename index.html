Time of Day

A JavaScript Project by KATIE LE

Greeting goes here

The way that this page works is, when you come at specific times of the day, the background color and image would change according to that time.

For example, if a user comes to the page between 5am and 12pm, that is considered morning time, so that will in turn show a orange background color with a bright yellow image with a woman, what looks like to represent a sun.

If a user comes between the time of 12pm to 6pm, that is considered afternoon, so the background color will change to be green with the image to be changed to a woman wearing green clothing as well.

If a user comes between the time of 6pm to 11pm, it is considered to be evening, so the background color would change to red and image to be of a woman with smoke and fire surrounding her.

If a user comes between the time of 11pm to 5am, the time is considered to be night, the background color would change to be purple with the image changed to be purple as well.
